What The Routes Do

Users

HTTP Method PATH Description
GET /users This path with GET method will get all users
GET /users/:userID This path with the GET method will get a specific user using the ID as a parameter to get
POST /users This path with the POST method will create a user, and hash the users password in the database
PATCH /users/:userID This path with the PATCH method will using the userID to identify which user you want to edit and will edit that specific user.
DELETE /user/userID This path with the DELETE http method will delete a user at that specific userID.

Products

HTTP Method PATH Description
GET /product This path with GET method will get all products
GET /products/:productID This path with the GET method will get a specific product using the ID as a parameter to get
POST /products This path with the POST method will create a product, and hash the product password in the database
PATCH /products/:productID This path with the PATCH method will using the productID to identify which product you want to edit and will edit that specific product.
DELETE /products/productID This path with the DELETE http method will delete a product at that specific productID.
